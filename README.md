# PTI Project

PTI(Pre-Trip Inspection) 데이터를 관리하고 대시보드를 제공하는 프로젝트입니다.

## 🚀 시작하기

이 프로젝트는 로컬 SQLite 데이터베이스와 Express 서버를 사용하여 데이터를 관리합니다.

### 📋 사전 요구사항

시작하기 전에 다음 프로젝트가 설치되어 있어야 합니다:
- [Node.js](https://nodejs.org/) (v18 이상 권장)
- npm (Node.js와 함께 설치됨)

### ⚙️ 설치 및 초기 설정

1.  **저장소 복제**
    ```bash
    git clone <repository-url>
    cd pti
    ```

2.  **의존성 패키지 설치**
    ```bash
    npm install
    ```

3.  **데이터 초기화**
    Excel 파일(`SAMPLE.xlsx`)의 데이터를 SQLite 데이터베이스로 변환해야 합니다.
    ```bash
    node import_data.mjs
    ```
    이후 서버가 처음 실행될 때 `src/initialData.json`을 기반으로 `pti.db` 파일이 생성됩니다.

### 💻 실행 방법

애플리케이션을 실행하려면 **백엔드 서버**와 **프론트엔드 개발 서버**를 모두 실행해야 합니다.

1.  **백엔드 서버 실행 (SQLite API)**
    ```bash
    node server.mjs
    ```
    서버는 `http://localhost:3001`에서 동작하며 데이터를 관리합니다.

2.  **프론트엔드 개발 서버 실행**
    별도의 터미널에서 다음을 실행합니다:
    ```bash
    npm run dev
    ```
    브라우저에서 `http://localhost:5173` 접속이 가능합니다.

## ✨ 주요 기능

- **📊 실시간 SQLite 데이터베이스**: 로컬 `pti.db` 파일을 통해 안전하고 빠른 데이터 저장 및 영속성을 보장합니다.
- **📊 실시간 대시보드**: PTI 현황을 시각화된 카드로 한눈에 파악할 수 있습니다.
- **📋 PTI 레코드 관리**: 검색, 필터링, 개별 편집 및 삭제 기능을 지원합니다.
- **📋 대량 데이터 추가 (Bulk Paste)**: 엑셀 등에서 복사한 데이터를 직접 붙여넣어 한 번에 여러 레코드를 생성합니다.
- **📈 통계 및 보고서**: 기간별 PTI 처리 현황 보고서를 제공합니다.
- **♻️ 휴지통 및 복구**: 삭제한 데이터를 복구하거나 영구 삭제할 수 있습니다.

## 📂 프로젝트 구조

- `docs/`: 요구사항 정의서 등 프로젝트 문서
- `src/`: React 프론트엔드 코드
- `server.mjs`: SQLite 연동을 위한 Express 백엔드 서버
- `pti.db`: 로컬 SQLite 데이터베이스 파일 (자동 생성)
- `import_data.mjs`: Excel 데이터를 JSON으로 변환하는 유틸리티
- `SAMPLE.xlsx`: 원본 데이터 샘플

## 📂 SQLite DB 파일 확인 방법

PC에서 SQLite `.db` 파일을 확인하는 가장 쉽고 대중적인 방법들을 추천해 드립니다.

1. **VS Code 확장 프로그램 (가장 추천)**
   현재 개발 환경이 VS Code라면 별도의 프로그램을 설치할 필요 없이 플러그인만 추가하는 것이 가장 편합니다.
   - **방법**: VS Code 왼쪽의 **Extensions**(Ctrl+Shift+X)에서 **"SQLite Viewer"**를 검색하여 설치하세요.
   - **장점**: `.db` 파일을 직접 클릭하면 엑셀처럼 표 형태로 바로 데이터를 볼 수 있습니다.

2. **웹 브라우저용 온라인 뷰어 (무설치)**
   프로그램 설치가 귀찮을 때 웹사이트에 파일을 드래그해서 바로 확인하는 방법입니다.
   - **사이트**: [sqliteviewer.app](https://sqliteviewer.app/)
   - **방법**: 웹사이트 접속 후 프로젝트 폴더의 `pti.db` 파일을 브라우저 창으로 끌어다 놓으면 됩니다.
   - **장점**: 설치가 필요 없고 인터페이스가 매우 깔끔합니다.

3. **DB Browser for SQLite (전용 소프트웨어)**
   전 세계적으로 가장 많이 쓰이는 SQLite 전용 무료 프로그램입니다.
   - **다운로드**: [sqlitebrowser.org](https://sqlitebrowser.org/)
   - **장점**: 데이터 조회뿐만 아니라 직접 수정, 엑셀로 내보내기/가져오기 기능이 매우 강력하고 직관적입니다.

> [!TIP]
> 지금 바로 확인해보고 싶으시다면 **1번(VS Code 확장 프로그램)** 혹은 **2번(웹 뷰어)**을 사용해 보시는 것을 가장 추천드립니다! 현재 진행 중인 프로젝트의 `pti.db` 파일을 열어보시면 `pti_records` 테이블에 저장된 데이터들을 확인하실 수 있습니다.
